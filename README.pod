This is Perl module B<App::Minify>.


=head1 INSTALLATION

App::Minify installation is straightforward. If your CPAN shell is set up,
you should just be able to do

    % cpan App::Minify

Download it, unpack it, then build it as per the usual:

    % perl Makefile.PL
    % make && make test

Then install it:

    % make install


=head1 DOCUMENTATION

App::Minify documentation is available as in POD. So you can do:

    % perldoc App::Minify
